## Features

- Modular window and engine architecture
- GLFW window abstraction
- Basic OpenGL context setup
## Goals

- Learn OpenGL rendering fundamentals
- Build a reusable engine architecture
- Experiment with window, input, and rendering systems

## Installation (Windows)

> Linux installation guide planned

1. Clone this repository with:

```bash
    git clone https://github.com/JJ0o0/PyJotaEngine
```

2. Create a VENV (Virtual Environment) and activate it:

```bash
    cd PyJotaEngine
    python -m venv venv
    .\venv\Scripts\activate
```

or do this automatically with an IDE like: Visual Studio Code.

3. Now, install the required dependencies:

```bash
    pip install -r requirements.txt
```

4. Run:

```bash
    python main.py
```
